Projeto Escultor - Parte 2
Documentação sobre a segunda parte do projeto Escultor desenvolvido na disciplina DCA1202 - Programação Avançada na UFRN no semestre 2020.2
Lista de componentes
Lista de classes, estruturas, uniões e interfaces com uma breve descrição:
 CcutBoxA classe CutBox é responsável por desativar o Voxel em um determinado intervalo em x, y ou z
 CcutEllipsoidA classe CutEllipsoid é responsável por desativar o Voxel em um determinado intervalo de x, y ou z
 CcutSphereA classe CutSphere é responsável por desativar o Voxel em um determinado intervalo de x, y ou z
 CcutVoxelA classe CutVoxel é responsável por desativar o Voxel em um determinado intervalo de x, y ou z
 CEscultorThe Escultor class Classe responsável por gerar o Escultor da figura 3D, dentro dessa classe encontram-se métodos que possibilitam o desenho da figura requerida pelo usuário sendo assim, ela se relaciona com a estrutura Voxel
 CFiguraGeometricaClasse FiguraGeometrica: Classe abstrata, ainda não foi definida completamente pelo sistema. A sua principal função é que as figuras sejam feitos de forma rápida e compreensível pelas outras classes
 CInterpretadorA classe Interpretador é responsável por fazer a leitura do arquivo "figura.txt" e gerar o arquivo .OFF
 CputBoxA classe PutBox é responsável gerar Voxels que representam uma caixa, ela realiza o preenchimento do espaço com base nos parâmetros de entrada fornecidos pelo usuário
 CputEllipsoidA classe PutEllipsoid é responsável gerar Voxels que representam uma elipse, ela realiza o preenchimento do espaço com base nos parâmetros de entrada fornecidos pelo usuário
 CputSphereA classe PutSphere é responsável gerar Voxels que representam uma esfera, ela realiza o preenchimento do espaço com base nos parâmetros de entrada fornecidos pelo usuário
 CputVoxelA classe PutVoxel é responsável ativar o Voxel, ela realiza o preenchimento do espaço com base nos parâmetros de entrada fornecidos pelo usuário
 CVoxelA estrutura Voxel é responsável por gerar a matriz 3D de acordo com seus parâmetros
